## How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/sanjaysam410/Chat.git
```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Open your browser and go to (http://localhost:3000)
